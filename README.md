# IoT-Based Live Weather Monitoring Dashboard

A real-time IoT weather monitoring system built with ESP32 that collects environmental sensor data and displays it on a live web dashboard.

## Features

- Real-time data collection (temperature, humidity, pressure)
- Live web dashboard for data visualization
- Sensor communication via I2C and SPI
- MQTT protocol for data transmission
- Automatic data validation and error handling

## Hardware

- ESP32 microcontroller
- BME280 sensor (temperature, humidity, pressure)
- Power supply unit

## Tech Stack

- **Firmware:** C++, Arduino framework
- **Protocols:** I2C, SPI, MQTT, UART
- **Dashboard:** Node.js, React
- **Tools:** VS Code, Git, PlatformIO

## Getting Started

### Prerequisites

- ESP32 development board
- BME280 sensor
- VS Code + PlatformIO extension
- MQTT broker (e.g. HiveMQ)
